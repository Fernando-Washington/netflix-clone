Trabalho Semestral

Desenvolva um site (ou replique algum já existente) que contenha, no mínimo, os itens abaixo:

  • 1 página utilizando GRID;

  • 1 página utilizando Flexbox;

  • Media Queries aplicadas em todas as páginas (quando possível, configure o layout com Media Queries);

  • 1 página com um formulário HTML formatado com CSS;

  • 1 página inicial com links para as demais páginas;

  • Caso optem por replicar um site já existente: não serão aceitos sites com pouco conteúdo ou com pouca aplicação dos conceitos listados acima, exemplo: Google.

  • Utilizem o máximo de CSS possível e preferencialmente externo.

  • Caso seja identificado o uso de IA ou trabalhos idênticos, a nota será zero.

  • Será avaliada a correta utilização do HTML e CSS. Não adianta apenas funcionar, o código deve estar correto.

  • O trabalho deve ser apresentado ao professor na próxima aula. Vocês deverão explicar o que fizeram e como fizeram (mostrando o código HTML e CSS).

  • Utilizem HTML e CSS puros. É proibido o uso de bibliotecas e frameworks. Lembrem-se de que a disciplina é de HTML e CSSfuturamente, terão disciplinas específicas para utilizar frameworks e bibliotecas.
  • Não serão aceitos trabalhos fora do prazo.
  • Não serão aceitos trabalhos sem a explicação.
  • O trabalho é individual.
